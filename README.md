# Psychic Game

A fun "guess the letter I'm thinking of" game.

## Built With
* HTML5
* CSS3
* JavaScript

## How to Use:
Users have 9 chances to guess the randomly selected letter for each round.

This app is a fun introduction to some basic JavaScript logic.
